G5 VPN 客户端 — Windows 测试安装包
版本: 0.1.0

【安装】
1. 解压本 ZIP 到任意目录（路径不要含特殊字符）
2. 双击「安装.bat」
3. 从桌面「G5 VPN」快捷方式启动

【卸载】
双击「卸载.bat」

【说明】
- 默认安装到: %LOCALAPPDATA%\Programs\G5Client
- 系统代理模式：普通用户即可
- TUN 全局模式：需右键 g5_client.exe → 以管理员身份运行
- 面板地址: https://g5.lulunet.cc

【目录文件】
- g5_client.exe     主程序
- data\             运行资源（含 sing-box / wintun）
- 安装.bat / 卸载.bat
