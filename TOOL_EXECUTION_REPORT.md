## ✨ 可直接使用的功能

### 1. 程序运行
```bash
./target/release/rsbox version
./target/release/rsbox check -c config.json
./target/release/rsbox run -c config.json
```

### 2. Docker 部署
```bash
docker-compose up -d
```

### 3. 一键安装
```bash
# Linux/macOS
curl -fsSL https://raw.githubusercontent.com/.../install.sh | bash

# Windows
iwr -useb https://raw.githubusercontent.com/.../install.ps1 | iex
```

### 4. 系统服务
```bash
./scripts/generate-service.sh
sudo systemctl start rsbox
```

### 5. 性能分析
```bash
./scripts/profile.sh
cargo bench
```
